## 描述

在 MCP MVP(v0.1/v0.2)稳定后,规划把 UIProbeWindow 视觉层从 IMGUI 迁移到 UI Toolkit。本任务为占位/前置规划,不写实现代码。

规划范围(基于 UIToolkitWorkbenchPlan.md):
1. 强制 IMGUIContainer 探针 spike:验证 layout/focus/repaint 在 UI Toolkit + IMGUIContainer 混排下的可行性,作为迁移前置门槛。
2. UI Toolkit Shell 骨架:工作台外壳 + Tab 容器;状态->UI 单向刷新机制(§7.1)。
3. 首个试点 Tab:以 Indexer Tab(已在 M2 完成 Service 化)为试点迁移到 UI Toolkit,黄金样本保证行为零变化。
4. 双栈退出标准(§9.1):IMGUI 与 UI Toolkit 并存期间的退出判据。
5. 锁定 Unity 2022.3 LTS+ 时序在 MCP MVP 之后。

产出:更新 Docs 下迁移任务清单,经用户确认后用 /amend 展开。

依赖前提:UI 视觉层迁移必须在数据层已 Service 化之后(M2),且不影响 MCP 链路。
